Imperia

Imperia adalah bot Discord open-source serbaguna dengan serangkaian fitur ekstensif yang dipisahkan menjadi beberapa modul. Cakupan Imperia yang direncanakan masih dalam tahap awal. Ini mencakup melakukan tugas-tugas seperti administrasi server, moderasi mudah dengan sistem otomatis, multibahasa, berbagai pilihan hiburan, pengumpulan informasi, sistem ekonomi, dll.

Imperia berasal dari bahasa Latin; Biasanya feminin, dan berasal dari kata Impĕro yang berarti perintah, kekuasaan, atau kekaisaran.

Imperia dikembangkan dan dibangun menggunakan Discord.js sebagai library utama berinteraksi dengan Discord API dengan Sapphire sebagai framework diatas Discord.js. Nx digunakan sebagai alat teknis untuk mengelola monorepo dan membangun proyek secara bersamaan. Next.js digunakan untuk membuat dashboard web yang akan digunakan untuk mengelola bot, dengan shadcn/ui sebagai styling. PostgreSQL dengan Drizzle ORM sebagai database.
