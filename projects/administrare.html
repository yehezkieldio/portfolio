Administrare

Administrare (/ad.mi.nisˈtra.re/) adalah sebuah aplikasi berbasis website yang bertujuan untuk mempermudah pengelolaan data internal seperti laporan keuangan, inventaris kantor, dan arsip catatan sebuah organisasi. Administrare atau Inventory Management adalah aplikasi website yang dikembangkan secara internal untuk Dinas Ketenagakerjaan dan Transmigrasi Balikpapan, sebagai proyek magang saya di sana.

Asal nama Administrare berasal dari bahasa Latin yang berarti "mengelola" atau "mengurus". Nama ini dipilih karena aplikasi ini bertujuan untuk membantu pengelolaan data internal sebuah organisasi.

Administrare dikembangkan dan dibangun berdasarkan stack MERN (MongoDB, Express, React, Node) dengan penggunaan teknologi dan library: Next.js sebagai frontend, Nest.js dan FastAPI sebagai backend, Chakra UI dan Redux sebagai state management dan user interface, dan MongoDB sebagai database.

Kode sumber tersedia di GitHub.
